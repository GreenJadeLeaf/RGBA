Target Transfer Color And Transfer Alpha
########################################

.. figure:: /images/transfer_entry.png
    
    The options Transfer Color and Transfer Alpha are in the target options

Transfer Color
    Transfer Color allows to bake or not the source color channels to the target color channels.

Transfer Alpha
    Transfer Alpha allows to bake or not the source alpha channel to the target alpha channel.

.. note::
    - The source color channels cannot change the target alpha channel.
    - The source alpha channel cannot change the target color channels.
    - The source alpha channel can influence the color channels with the *Do Alpha Blend* option in :ref:`Save RGBA Settings <transfer_list/transfer_entry_settings:Save RGBA Settings>`.
