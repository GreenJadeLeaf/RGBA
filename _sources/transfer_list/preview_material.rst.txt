Preview Material
################

.. |blender_icon_material| image:: /images/blender_icon_material.png
    :height: 18px

When setting up the bake, it can be useful to preview what the source or target look like.
It allows to quickly preview and unpreview a bake setup without needing to go change the shader node tree.

.. figure:: /images/transfer_entry.png
    
    The preview material operator has multiple locations, each with a specific context in a transfer entry

The preview material operator |blender_icon_material| is located on the right of each material subentry.

.. admonition:: Usage:

    - To preview a material, click on the material icon on the right of a material subentry.
    - To unpreview a material, click again on the same material icon on the right of a material subentry.

There are multiple modes to preview a material:

- Color output
    Preview the color output
- Alpha output
    Preview the alpha output
- Transparent BSDF
    Preview the color output using the alpha output as a factor to mix between a Transparent BSDF and the color output

.. note::
    It is best to unpreview any material that is currently previewing before launching the bake operator when using the source material modes :ref:`Principled BSDF <transfer_list/source_material_modes:Principled BSDF>` or :ref:`Material <transfer_list/source_material_modes:Material>`.
