RGBA Bake
#########

.. _operator_rgba_bake:

The operator RGBA Bake launches the baking process.
All selected transfer entries will be processed.
A progress bar is shown in the bottom-right corner.

This operator can be accessed in multiple locations according to the preferences settings.

- In the Scene Render properties
- In the Node Editor properties
- In the Sidebar of the 3D Viewport

.. note::
    The operator is running in modal mode to allow the interface to update the progress bar.
    Any interaction with Blender when the RGBA Bake operator is running might cause crashes and unexpected errors.
    The RGBA Bake operator shows a message of completion when it has finished to process and can be cancelled with the Escape *ESC* key.
